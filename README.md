# t-bank

> 荣大科技-智慧投行

## Build Setup

``` bash
# 安装依赖 install dependencies
npm install

# 运行 serve with hot reload at localhost:8080
npm run dev

# 构建 build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
